# Setting up Splunk:
-I installed Splunk Enterprise on the Windows server, configured it to receive logs from both local and remote hosts on the domain.
- To parse logs from the endpoint to the indexing server (the Windows server), under Forwarding and Receiving, I enabled listening on port 9997, which is the default port that Splunk uses to receive logs from Splunk Universal Forwarder.
