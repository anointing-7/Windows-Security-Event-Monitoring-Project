# Windows-Security-Event-Monitoring-Project

In this project, I set up two virtual machines: an Active Directory Domain Controller and a Windows 10 endpoint. I configured log monitoring on each virtual machine, generated Windows security events, and forwarded the resulting logs to a SIEM platform. Using the SIEM, I searched, analyzed, and investigated selected security events to simulate a Security Operations Center (SOC) workflow.

## Tools Used:
- Windows Server 2022 (Active Directory Domain Services)
- Windows 10 Endpoint
- Splunk Enterprise (SIEM)
- Splunk Universal Forwarder
- Sysmon
